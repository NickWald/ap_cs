# bear class

```java
bear mama = new bear();
bear baby = new bear();
bear papa = new bear();

mama.hug(baby);
mama.hug(papa);
```

**MAMA:**
hugCount = 2;
**BABY:**
hugCount = 1;
**PAPA:**
hugCount = 1;

totalHugs = 2; (number of hug statements) (keeps track of number of times method is called)
A pair is one hug
